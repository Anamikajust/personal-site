About

I'm a PhD candidate in the Department of Mathematics at the University of Louisiana at Lafayette, working with Dr. Yangwen Zhang on fast, memory-efficient methods for time-fractional partial differential equations. Fractional-order models capture memory effects — sub-diffusion, viscoelasticity, anomalous transport — but that same memory makes them expensive: classical schemes must store the entire solution history at every time step.

My dissertation work builds on the ISVD-compressed L1 finite element framework, tightening the compression error bound from O(n·tol) to O(√n·tol) — a change that lets these solvers keep far less in memory without giving up accuracy.

Outside the dissertation, I teach undergraduate mathematics as a graduate teaching assistant during the academic year, and work full-time on research through the summer. My earlier research spans analytical methods for nonlinear fractional evolution equations and computational biology.

Research

Time-fractional PDEs model systems with memory — the solution at every past moment influences the present. Discretizing them (e.g. with L1-type schemes) means storing that entire history, which grows the memory footprint linearly with the number of time steps, n. The ISVD (incremental SVD) framework compresses that history on the fly. My contribution tightens the bound on how much compression error that introduces.

Compression error bound

O(n·tol) → O(√n·tol)
Prior bound — grows linearly with n This work — grows with √n
01 / Theory

Sharper ISVD compression bounds

Improved the incremental-SVD compression error bound for the L1-FEM history matrix from linear to square-root growth in the number of time steps, using a discrete fractional Grönwall inequality applied to the unsquared error norm — the step that preserves the √n improvement rather than losing it to squaring.

02 / Method

ISVD-compressed L1-FEM

Builds on Dr. Zhang's ISVD-compressed L1 finite element framework for time-fractional sub-diffusion equations, which compresses the solution history on the fly instead of storing it in full.
